## 描述

用于HTTP 提交的内容验证的中间件和zend-mvc监听器。
内容验证使用 `zend-inputfilter`组件.

* `ContentValidationMiddleware` PSR-7 中间件
* `ContentValidationListener` zend-mvc 监听器
