## 描述

用于 PSR-15 中间件或 laminas-mvc 的内容验证。
内容验证使用 [`laminias/laminas-inputfilter`](https://github.com/laminas/laminas-inputfilter)组件.

* `ContentValidationMiddleware` PSR-7 中间件
* `ContentValidationListener` laminias-mvc 监听器
